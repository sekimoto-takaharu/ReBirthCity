# RE:BIRTH CITY（リバース・シティ）
## 〜 100年後の「お疲れ様」をデザインする、都市経営シミュレーション 〜

## 1. プロジェクト・ビジョン
日本の市区町村が直面する「少子高齢化・財政難」という現実の課題を、
「終活（人生の整理と継承）」という戦略的アクションで解決し、
自分が去った後の世界を「明るく前向きな社会」にリフォームする。

## 2. コア・コンセプト
- **現実連動:** 実在する自治体の統計データを初期値として使用。
- **終活の資産化:** 遺言、空き家、相続、知恵の継承を「街の発展リソース」に変換。
- **未来観測:** タイムスライダーで30年・100年後の「自分の死後」をシミュレート。

## 3. 技術スタック (Technical Stack)
- **Framework:** Flutter (iOS/Android対応)
- **Language:** Dart
- **State Management:** Riverpod
- **Backend:** Firebase (Authentication, Firestore)
- **Data Source:** e-Stat (政府統計名寄せデータ)
